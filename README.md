🚀 Built an AI-Powered Website Copy Generator for Local Businesses

Excited to share my latest project developed as part of my learning journey in **AI & Prompt Engineering** 👇

💡 **Problem:**
Many coaching institutes and local businesses struggle with:
❌ Poor website copy
❌ Weak value proposition
❌ Low conversions

Hiring professional copywriters is expensive — and most don’t know what to write.

---

🧠 **Solution:**
I built an **AI Website Copy Generator** that creates high-converting, SEO-optimized content using **Gemini API + Prompt Engineering**

---

⚙️ **Key Features:**
✅ Generates complete website copy (Hero, About, Courses, CTA)
✅ SEO keyword integration for better visibility
✅ Structured JSON output for clean UI rendering
✅ Section-wise UI using Streamlit
✅ Downloadable content (ready to use)
✅ Robust error handling for real-world reliability

---

🛠️ **Tech Stack:**
🔹 Python
🔹 Streamlit
🔹 Google Gemini API
🔹 Prompt Engineering
🔹 JSON Parsing & Regex

---

🎯 **What I Learned:**
🔹 Designing structured prompts for real-world use cases
🔹 Handling inconsistent AI outputs (production-level thinking)
🔹 Building user-friendly AI tools
🔹 Applying SEO strategies in AI-generated content
